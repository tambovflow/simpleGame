Это простая консольная игра, которая угадывает выбранное вами число.
Схема игры построена на популярном карточном фокусе, только вместо карт решено было использовать простой набор чисел.

Для запуска просто скомпилируйте .java файл и запустите его в консоли.
В интерфейсе присутствуют кирилические символы, так что если ваша консоль их не поддерживает, то земля ей пухом, господа.
